# 模拟电子线路EDA

## 课程基本情况

- 学分：1
- 期末考核方式：考查
- 成绩构成：
  - 设计报告（50%）
  - 课堂表现（50%）

## 学习内容

大三部分专业的必修课程。需要在一周内通过Multisim软件进行单级放大电路的设计与仿真、阶梯波发生器的设计与仿真、负反馈放大电路的设计与仿真、阶梯波发生器的设计与仿真。
