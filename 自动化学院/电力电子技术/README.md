# 电力电子技术

## 课程基本情况

- 学分：3
- 期末考核方式：考试
- 成绩构成：
    - 闭卷笔试（70%）
    - 实验报告（10%）
    - 平时成绩（20%）

## 学习内容

电能变换和控制。学习各种功率元件，基本的电能变换电路的工作原理、波形等。
